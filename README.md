# What's your current weather? ..

A simple app which tells you your weather or 5-day forecast.

## Technologies used
  - ExpressJS
  - Babel
  - HTML5 Geolocation
  - ReactJS/Redux
  - Google Maps API
  - DarkSky API
  - Bulma
