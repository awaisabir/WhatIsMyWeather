# What's your current weather? ..

A simple app which tells you your weather or 5-day forecast.

## Technologies used
  - ExpressJS
  - Babel
  - HTML5 Geolocation
  - Javascript (ES6)
  - Google Maps API
  - DarkSky API
  - Bulma

## Current Status
Vanilla JS version (v1) completed.
