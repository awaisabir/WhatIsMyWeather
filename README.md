# What's your current weather? ..

A simple app which tells you the current weather conditions or weekly forecast based on your location.

## Technologies used
  - ExpressJS
  - Babel
  - HTML5 Geolocation
  - ReactJS/Redux
  - Google Maps API
  - DarkSky API
  - Bulma
